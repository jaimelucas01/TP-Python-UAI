## 📖 Descripción

Smart Data Analyzer es una aplicación web desarrollada con Streamlit que permite realizar un análisis exploratorio de datos a partir de archivos CSV proporcionados por el usuario.

La aplicación integra técnicas de Análisis de Datos con Pandas y Machine Learning mediante el algoritmo K-Means de Scikit-Learn.

## 🚀 Funcionalidades

* Carga de archivos CSV.
* Visualización del conjunto de datos.
* Análisis automático del dataset.
* Identificación de tipos de datos.
* Detección de valores nulos.
* Estadísticas descriptivas.
* Gráficos interactivos.
* Agrupamiento de datos mediante Machine Learning (K-Means).

## 🛠 Tecnologías utilizadas

* Python 3
* Streamlit
* Pandas
* Plotly
* Scikit-Learn

## ▶️ Ejecución local

Instalar las dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar la aplicación:

```bash
streamlit run app.py
```
